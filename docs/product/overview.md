# Product Overview

## Resumen
Gimnasia es una app B2C multiplataforma (web + movil) orientada a seguimiento de entrenamiento, dieta y medidas corporales, con capacidades de IA opcionales mediante BYOK (Bring Your Own Key).

## Alcance v1
- Publico: usuarios de 18+ (autodeclaracion).
- Tipo de lanzamiento: beta privada.
- Objetivo de uso: 2 MAU.
- Modelo de negocio inicial: gratis.
- Modulos v1:
  - Entrenamiento
  - Dieta
  - Medidas
  - Objetivo activo
  - Chat IA
  - Generacion multimedia para ejercicios

## Comportamiento BYOK
- BYOK es opcional.
- Si usuario no configura API key:
  - Tracking funciona normalmente.
  - Funcionalidades IA quedan deshabilitadas en UI.

## KPI de salida beta
- Sync exitoso >= 98%.
- Sesiones sin crash >= 99%.
