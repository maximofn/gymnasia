# Specs V1

Esta carpeta contiene las especificaciones formales de v1.

## Documentos
- `v1-product-spec.md`: especificacion funcional de producto.
- `v1-technical-spec.md`: especificacion tecnica de arquitectura y componentes.
- `v1-api-spec.md`: contrato de API REST de v1.
- `v1-data-spec.md`: especificacion de modelo de datos y reglas.
- `v1-non-functional-spec.md`: especificacion no funcional (seguridad, rendimiento, operacion).
- `functional/README.md`: indice de specs funcionales por modulo.
- `functional/training-functional-spec.md`: spec funcional de entrenamiento.
- `functional/diet-functional-spec.md`: spec funcional de dieta.
- `functional/measurements-functional-spec.md`: spec funcional de medidas.
- `functional/goals-and-ai-functional-spec.md`: spec funcional de objetivos y BYOK.
