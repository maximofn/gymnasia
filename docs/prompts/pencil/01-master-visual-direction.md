# Prompt 01 - Direccion Visual Maestra

Disena un sistema visual para una app de gimnasio moderna y premium llamada “Gimnasia”. Estilo limpio, energetico, profesional, con alto contraste y enfoque mobile-first. Crear variables de color, tipografia, espaciado, radios, sombras y estados (hover/pressed/disabled). Incluir componentes reutilizables: botones, inputs, cards, tabs, chips, barra inferior, modales y toasts. Debe verse consistente en web y movil.
